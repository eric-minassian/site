# Site

Free, managed personal-site platform. Write markdown, pick a template, get a fast static site on a CDN.

## Structure

```
infra/    CDK infrastructure (pipeline, stacks, config)
docs/     Design docs, requirements, plan
```

## Setup

```sh
pnpm install
```

## Commands

```sh
pnpm test        # run all tests
pnpm typecheck   # type-check all packages
pnpm build       # build all packages
```
